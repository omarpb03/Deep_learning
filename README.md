# Deep Learning Implementations 🧠

Este repositorio contiene una colección de implementaciones prácticas de arquitecturas de Deep Learning, enfocadas en la resolución de problemas complejos mediante modelos predictivos y clasificadores avanzados.
Los notebooks están organizados por el tipo de arquitectura y técnica implementada:

### 1. Introducción y Fundamentos
- **Fundamentos de ML:** Introducción al aprendizaje máquina, manejo de `train_test_split` y validación cruzada.
- **K-Nearest Neighbors (KNN):** Implementación básica de algoritmos de clasificación.

### 2. Multi-Layer Perceptron (MLP)
- **Implementaciones en TF/Keras:** Configuración de densas capas ocultas, funciones de activación y optimizadores para problemas de regresión y clasificación.

### 3. Redes Neuronales Convolucionales (CNN)
- **Clasificación de Imágenes:** Implementación de CNNs para el dataset MNIST y detección de patrones espaciales.
- **Operaciones de Convolución:** Notebooks detallados sobre el funcionamiento de filtros y capas de pooling.

### 4. Redes Neuronales Recurrentes (RNN)
- **Modelado Secuencial:** Ejemplos basados en el libro de *Hands-On Machine Learning* de Aurélien Géron para el tratamiento de series temporales y secuencias.

### 5. Frameworks Avanzados (PyTorch)
- **PyTorch Tutorial:** Implementación de tensores y flujos de trabajo optimizados en el ecosistema de PyTorch.

## 📈 Aplicaciones destacadas
- **Procesamiento de MNIST:** Alta precisión en clasificación de dígitos escritos a mano.
- **Arquitecturas UNIR:** Proyectos realizados bajo estándares académicos rigurosos.
